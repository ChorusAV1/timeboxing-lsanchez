# Instalación

## Instalar frontend

1. Entra a la carpeta del fronend

    ```bash
    cd timeboxing-frontend
    ```

2. Instala con npm

    ```bash
    npm i
    ```

3. Ejecuta en dev

    ```bash
    npm run dev
    ```
